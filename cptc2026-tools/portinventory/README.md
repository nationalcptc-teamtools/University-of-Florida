# PortInventory
Take inventory of all ports on a network in minutes!

# Prerequisites
 * Python 3.10 or newer
 * Python Requests library, version 2.19 or later

# Setup
After setting up Planka in the `planka` subdirectory, add administrative credentials in `plankaconfig.ini`. Then, run
```
python plankainit.py
```
to add the users in `plankausers.txt` (one per line) to the database.
